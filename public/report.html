REPORT

1. In order to file a complaint with a state medical board when personal or familial harm has been caused by medical decision making of a physician, you will need to go to the physician’s state medical board official website and file a complaint electronically. Some states require that you create an account first. The state may not accept an anonymous complaint.

2. None of your personal information will be collected on this web site unless you choose to use the optional intake form.

3. The information you may need to file your complaint with the board are:

Treatment Date
Adverse Reaction Discovery Date
Physician Name
License Number (this can commonly be found at the complaint site)
Facility/Practice Name
Medical Specialty
Injury/Averse Reaction Description (you are encouraged to personalize this template which is provided)
Medical Documentation (of your injury evaluation)

4. Pease be aware that false reporting may result in legal consequences.
